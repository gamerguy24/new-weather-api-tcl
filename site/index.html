🌩️ NEXRAD Radar — static client-side API

This host serves one thing: nexrad.js. It's a JavaScript module you import from a web page — it lists radar sites and decodes live NEXRAD Level II from AWS Open Data entirely in the browser. There is no server and no UI here.

Use it

import * as NEXRAD from "./nexrad.js";

const sites = await NEXRAD.listSites();            // [{ code, name, lat, lon }]
const scan  = await NEXRAD.getScan("KTLX");        // newest scan for a site

scan.metadata();     // { site, key, scanTimeUTC, siteLat, siteLon, bounds, ... }
scan.toDataURL();    // rendered base-reflectivity PNG (data URL)
scan.imageData;      // raw pixels (ImageData) if you want to draw it yourself

API

CallReturns
listSites()sites reporting today: [{ code, name, lat, lon }]
latestKey(site)newest S3 key, or null
getScan(siteOrKey, opts)a Scan object (metadata + toDataURL() / toBlob() / imageData)
decode(arrayBuffer)low-level decoded radials
render(decoded, {size})RGBA raster + lat/lon bounds

Full reference: README.md. Source: nexrad.js, js/, vendor/bz2.js. A static host serves files, not computed responses, so this is a JavaScript API you import — not a curl-able endpoint.
